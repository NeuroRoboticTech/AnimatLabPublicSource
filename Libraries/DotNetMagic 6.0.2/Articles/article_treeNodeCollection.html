NodeCollection Overview
The NodeCollection class is just to hold a group of sibling Node instances. An instance is used as to hold the root level nodes in a TreeControl and is used in a Node instance to store the collection of child nodes. You should also read the article that describes the Node class by using the following link Node article and the main TreeControl article using this link TreeControl article.

The following sections show how to use Node class and describe the more important properties, methods and events for controlling the Node. This is not however an exhaustive list and so the help document should be consulted for the definitive list of all functionality.

Collection
Add()
Appends a Node instance to the end of the collection.

AddRange()
Appends an array of Node instances to the end of the collection.

Remove()
Remove an existing Node instance from the collection.

Insert()
Add a Node instance at a given index position.

Contains()
Returns true if the collection contains the Node instance.

IndexOf()
Returns the index position of the Node instance.

this[]
Returns the Node instance at the specified index position.

Navigation
ParentNode
A read-only property that returns a reference to the Node instance that contains this collection.

TreeControl
A read-only property returning the reference to the TreeControl that this instance is a part of. Note that this can return null when a node is created but has not yet been added to a part of the tree hierarchy for a TreeControl.

GetFirstNode()
Returns the first Node instance in the collection or null if the collection is empty.

GetFirstDisplayedNode()
Returns the first Node instance in the collection that is visible for display or null if none can be found.

GetLastNode()
Returns the last node in the child hierarchy. It ignores the visible state of any node and returns the last node regardless of its current state.

GetLastDisplayedNode()
Unlike the previous method this one will take into account the visible state of individual nodes. Therefore it will return the last node in the child hierarchy that is visible.

VisibleCount
Return a count of the total number of nodes in the collection that are defined as visible for display.

FirstVisibleIndex
Returns the index position of the first node in the collection that is visible for display.

LastVisibleIndex
Returns the index position of the last node in the collection that is visible for display.

Size
A read-only property that returns the last calculated size needed to draw the node collection.

Bounds
A read-only property that returns the last calculated bounding rectangle for drawing the node collection.

ChildBounds
A read-only property that returns the last calculated bounding rectangle for the node collection that also includes all the child nodes as well.

IsSizeDirty
A read-only property that returns true when the size of the node needs calculating.

VC
An advanced property used to assign a specific ViewController implementation for handling all the drawing and behavior of the node collection instance. By default this is null and so the TreeControl defined ViewController will be used instead.

Events
Clearing
Generated when the collection is about to be cleared of all nodes.

Cleared
Generated after the collection has been cleared of all nodes.

Inserting
Generated when the collection is about to add or insert a new node.

Inserted
Generated after the collection has added or inserted a new node.

Removing
Generated when the collection is about to remove a node.

Removed
Generated after the collection has removed a node.
